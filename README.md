# SMT產線管理指揮台

整合SMT產線設備連線、程式管理、追溯與品質數據之專用產線管理系統。

- 原始系統：SMT產線管理系統（SMT Line Management System）
- 產業分類：智慧製造｜製造管理
- 使用部門：SMT製造工程、生產、品保
- 工作流程：程式下發 → 設備連線執行 → 製程/檢測數據採集 → 追溯資料綁定 → 產線績效分析
- AI 應用：AI整合SPI/AOI數據預測製程異常、智慧優化貼片程式參數
- 主要 KPI：SMT稼動率、首件合格率、產線異常停機時間
- 代表產品：Aegis FactoryLogix、Cogiscan、Siemens Opcenter Execution SMT（原Valor）
